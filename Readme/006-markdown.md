# Todo List
## 專案介紹
透過開發 Todo List 練習使用 JavaScript 建立互動式網頁功能， 並從使用者角度思考待辦事項管理流程，驗證功能是否符合預期需求， 同時培養問題分析、功能驗收與資料同步的實作能力
---

## 功能
- 新增待辦事項
- 刪除待辦事項
- 標記完成

---

## 開發過程

### 遇到的問題1

使用者新增待辦事項的刪除按鈕無法使用。

### 解決方法1

利用 Event Delegation 方法監聽列表區域，再抓取標籤等於button的物件刪除。

### 遇到的問題2

新增完成按鈕會讓同一層中有2個標籤等於button的，點擊任意一個按鈕都會偵測為刪除。

### 解決方法2

將監聽標籤改成監聽class name。

### 遇到的問題3

新增按鈕只監聽click事件。

### 解決方法3

將function和監聽事件分開撰寫，並再寫一個監聽事件偵測keydown事件。

### 遇到的問題4

輸入空白依舊可以新增待辦事項。

### 解決方法4

利用 trim() 偵測並去除前後空白，再寫if條件是回傳空字串。

### 遇到的問題5

要完成和已完成的樣式有所區分，但新增之無法再切換。

### 解決方法5

使用 toggle 切換狀態。

### 遇到的問題6

資料只存在在畫面上，更新就會消失。

### 解決方法6(解決中)

先加入陣列，將輸入的資料存到陣列中，
