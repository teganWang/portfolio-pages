專題名稱:003-shopping-cart-counter
專案詳情:使用 JavaScript 製作商品數量加減功能，透過 DOM 操作與事件監聽更新畫面數值，並限制商品數量不可小於 0。
    遇到的問題(1):從畫面取得的數值為字串型別，直接與數字相加時會出現字串串接問題。
    解決方法:另外設一個變數等於0，計算過後再帶回變數和顯示框
      也可以:使用 Number() 將取得的字串轉為數字型別後再進行計算。
    遇到的問題(2):當數量減少至 0 時會更改警示文字，但數值恢復後，原本的提示文字無法正常還原。
    解決方法:先儲存原始提示文字內容，待數值恢復正常後重新帶回原始文字。
學習收穫:
    1.理解 JavaScript 中字串與數字型別轉換的差異
    2.練習使用 if 條件式處理畫面狀態判斷
    3.熟悉 DOM 操作與事件監聽的基本應用
